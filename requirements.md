# Lottery API

- sell lottery tcket
- update lottery ticket
- delete lottery ticket
- get all lottery tickets
- get ticket by id
- bulk buy (user can purchase multiple tickets)
- raffle draw

Ticket :

- number (unique)
- username
- price
- timestamp

Routes:

- /tickets/sell - create new ticket
- /tickets - find all tickets
- /tickets/t/:id - find ticket by id
